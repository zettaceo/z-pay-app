Roadmap Z-PAY

Features para se destacar da concorrência

Alta

Integração Z-PAY + Z-Finance

Conta Z-Finance como carteira para receber pagamentos. Saldo disponível para investimentos, empréstimos ou transferências. Ecossistema fechado.

Alta

Link-in-bio / Checkout 1 clique

Link único por merchant (pay.zetta.com/loja) com catálogo de produtos. Ideal para influenciadores e freelancers.

Alta

Recorrência via PIX

Assinaturas com cobrança automática mensal via PIX. Alternativa ao cartão recorrente.

Alta

Antecipação com taxa fixa

Antecipe vendas a cartão com taxa única (ex: 1,5% a.m.). Sem spread oculto.

Alta

Relatórios PDF

Exportar PDF com branding Z-PAY. Relatórios agendados por e-mail.

Alta

Dashboard tempo real

WebSocket para atualização live. Notificações push quando pagamento aprovado.

Novo

PIX Automático (Recorrência BACEN)

Débito automático via PIX autorizado pelo pagador. Lançado pelo BACEN em junho/2025 — alternativa ao cartão recorrente sem intermediários.

Novo

Bolepix (Boleto + PIX)

Boleto com QR Code PIX embutido — o pagador escolhe como pagar. Mandatório BACEN a partir de novembro/2026. Quem implementar antes, sai na frente.

Média

Split em tempo real

Divisão automática entre vendedor, afiliado e plataforma. Regras por produto.

Média

PIX parcelado

Parcelamento via PIX com taxas transparentes. PIX crédito.

Média

White-label

Checkout com logo, cores e domínio do cliente. E-mails personalizados.

Média

API-first + SDKs

SDKs Node, Python, PHP, React. Documentação Swagger. Sandbox gratuito.

Média

Anti-fraude avançado

Score por transação. Regras customizáveis. Blacklist. Relatório de suspeitas.

Média

Gestão de chargebacks

Lista de disputas. Upload de documentos. Timeline. Alertas de prazo.

Média

Notificações WhatsApp Business

Confirmação de pagamento instantânea via WhatsApp. Lembretes de boleto, alertas de valores altos e resumos semanais automáticos.

Média

Integração ERP (Conta Azul, Omie, Bling)

Sync automático com os maiores ERPs do Brasil. Dados formatados para DRE, conciliação contábil e SPED.

Longo

Open Banking

Conexão com contas bancárias. Pagamentos via débito em conta (DDA).

Longo

BNPL (Buy Now Pay Later)

Parcelamento 3x sem juros para o comprador. Custo no merchant ou compartilhado.

Longo

Marketplace nativo

Múltiplos vendedores em uma loja. Split automático. Relatórios por vendedor.

Longo

Multi-currency

Dashboard de câmbio. Conversão automática. Carteira multi-moeda.

Longo

Integrações

Zapier, Make. Webhooks avançados. Retry configurável.

Longo

Suporte

Chat in-app. Base de conhecimento. Status page (uptime).

2026

Split Tributário (Reforma CBS/IBS)

Dedução automática de impostos no momento da transação. Preparação para o split payment tributário que entra em vigor em 2026. Dois regimes coexistem até 2033.

Longo

PIX por Aproximação (NFC)

Pagamento PIX via NFC no terminal do merchant. Lançado pelo BACEN em fev/2025 — integração com POS físico.
